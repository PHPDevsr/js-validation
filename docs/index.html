js-validation Demo

Vanilla JS form validation (jquery-validation style). Rules can come from JS options or data-rule-* attributes.

Sample Output (when validation fails)

When a field fails validation, the following HTML is produced:

<div class="form-group">
  <label for="email">Email</label>
  <input id="email" name="email" type="text"
    class="is-invalid"
    aria-invalid="true"
    invalid="true"
    data-jsv-message="Please enter a valid email address." />
  <span class="invalid-feedback"
    data-jsv-error-for="email">Please enter a valid email address.</span>
</div>

When validation passes

<div class="form-group">
  <label for="email">Email</label>
  <input id="email" name="email" type="text"
    aria-invalid="false" />
</div>
